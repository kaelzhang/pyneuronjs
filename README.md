# pyneuron

Python utilities and middleware for neuron.js

## Install

```sh
$ pip install pyneuron
```

## Usage

```py
from pyneuron import Neuron

neuron_instance = Neuron(
  dependency_tree=dependency_tree,
  decorate=decorator,
  path='mod'
)
```
